# Junior Software Developer - Melbourne

## Company: TechInnovate Solutions

## Location: Melbourne, VIC (Hybrid - 3 days office, 2 days remote)

## About the Role

We are seeking a motivated Junior Software Developer to join our growing development team. This is an excellent opportunity for a recent graduate to work on real-world projects and grow their career in software development.

## Responsibilities

- Develop and maintain web applications using modern JavaScript/TypeScript frameworks
- Write clean, maintainable, and well-documented code
- Collaborate with senior developers on complex features
- Participate in code reviews and agile ceremonies
- Debug and troubleshoot issues in existing applications
- Contribute to technical documentation

## Required Skills

- Bachelor's or Master's degree in Computer Science, IT, or related field
- Strong understanding of JavaScript/TypeScript
- Experience with React or similar frontend frameworks
- Knowledge of RESTful APIs
- Familiarity with version control (Git)
- Good problem-solving skills
- Excellent communication skills

## Nice to Have

- Experience with Node.js or Python backend development
- Familiarity with cloud platforms (AWS, Azure, or GCP)
- Knowledge of databases (SQL and NoSQL)
- Experience with Agile/Scrum methodologies
- Personal projects or contributions to open source

## What We Offer

- Competitive salary package
- Professional development opportunities
- Flexible work arrangements
- Modern office in Melbourne CBD
- Collaborative and inclusive team culture

## How to Apply

Submit your resume and a brief cover letter explaining why you're interested in this role.
